# 企业官网规格文档

## 1. 项目概述
- **项目名称**: 科技企业官网
- **类型**: 企业展示网站
- **核心功能**: 公司展示、产品服务介绍、企业形象宣传
- **目标用户**: 潜在客户、合作伙伴、求职者

## 2. 技术架构
- **前端**: HTML5 + CSS3 + JavaScript
- **后端**: Python Flask
- **端口**: 前端 8080，后端 5000

## 3. UI/UX 规格

### 3.1 视觉设计
- **主色调**: 
  - 主色: #0A1628 (深蓝黑)
  - 辅助色: #1E3A5F (中蓝)
  - 强调色: #00D4AA (青绿)
  - 浅色: #F8FAFC (淡灰白)
- **字体**: 
  - 标题: "Noto Sans SC", sans-serif
  - 正文: "Noto Sans SC", sans-serif
- **间距**: 8px 基础单位
- **圆角**: 12px (卡片), 8px (按钮), 24px (大圆角)

### 3.2 布局结构
- 响应式设计: 桌面端 > 1024px, 平板 768-1024px, 移动端 < 768px
- 最大内容宽度: 1200px
- 固定导航栏高度: 72px

### 3.3 页面结构

#### 首页 (index.html)
1. **导航栏**: Logo + 导航链接 + 联系方式按钮
2. **Hero区域**: 大标题 + 副标题 + CTA按钮 + 装饰性图形
3. **数据展示**: 3列数字统计 (成立年限、客户数量、项目数量、满意度)
4. **服务介绍**: 4个服务卡片，hover动效
5. **客户评价**: 轮播或网格展示
6. **合作伙伴**: Logo墙
7. **底部**: 联系信息 + 快速链接 + 社交媒体

#### 产品/服务页 (services.html)
1. 顶部banner
2. 服务分类导航
3. 服务详情卡片 (6个服务)
4. 咨询表单

#### 关于我们页 (about.html)
1. 顶部banner
2. 企业简介
3. 时间线/发展历程
4. 团队成员展示
5. 企业价值观

### 3.4 动画效果
- **页面加载**: 元素渐入动画 (fadeInUp)
- **导航**: 滚动时背景变化 + 滚动阴影
- **卡片**: hover时轻微上浮 + 阴影增强
- **按钮**: hover时颜色过渡 + 轻微缩放
- **滚动**: 元素进入视口时触发动画 (Intersection Observer)
- **平滑滚动**: 锚点链接平滑滚动

### 3.5 交互设计
- 导航高亮当前页面
- 卡片可点击跳转
- 表单验证和提交反馈
- 返回顶部按钮
- 移动端 hamburger 菜单

## 4. 后端 API 规格

### 4.1 接口列表
- `GET /api/company` - 获取公司信息
- `GET /api/services` - 获取服务列表
- `GET /api/stats` - 获取统计数据
- `POST /api/contact` - 提交联系方式

### 4.2 数据格式
```json
{
  "company": {
    "name": "公司名称",
    "description": "公司简介",
    "established": "2015"
  },
  "services": [
    {
      "id": 1,
      "title": "服务名称",
      "description": "服务描述",
      "icon": "图标类名"
    }
  ],
  "stats": {
    "years": 9,
    "clients": 500,
    "projects": 1200,
    "satisfaction": 98
  }
}
```

- [ ] 页面在现代浏览器正常显示
- [ ] 响应式布局在三种屏幕尺寸正常
- [ ] 所有动画流畅 (60fps)
- [ ] 导航正常跳转
- [ ] 后端API正常返回数据
- [ ] 表单提交有反馈
